# AI-Powered Job Tracker & Resume Analyzer

A full-stack application for job seekers to track their applications and analyze their resumes using AI.

## Features
- **Kanban Board**: Track jobs through "Applied", "Interview", "Rejected", and "Offer" statuses.
- **AI Resume Analyzer**: 
  - PDF and Text file support.
  - Job Description matching with match scores.
  - Smart bullet point rewrites.
  - Skills gap analysis.
- **User Authentication**: Secure JWT-based login and registration.

## Tech Stack
- **Frontend**: React, Vite, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express, MongoDB, OpenAI API.

## Setup
1. Clone the repository.
2. Install dependencies for both `client` and `server`.
3. Add your `OPENAI_API_KEY` to the `server/.env` file.
4. Run `npm start` in the `server` directory and `npm run dev` in the `client` directory.
